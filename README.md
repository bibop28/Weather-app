# 🔥 Weather App

Weather App is a simple and modern web application for checking the weather in any city. It provides temperature, humidity, wind speed, and weather descriptions with a clean and responsive interface. The application is designed to be user-friendly and easy to navigate. 



---
## ✨ Key Features
- 🏙️ Search weather by city name
- 🌡️ Display temperature in Celsius
- 💧 Show humidity and wind speed
- 🌥️ Weather description with icons
- ⬆️ Press Enter or click the magnifying glass to search

---
## 🧭 Simple Workflow
1. Open the Weather App in your browser
2. Enter a city in the search input
3. Click the search button or press Enter
4. Weather information appears below the search bar

---
## 🎯 Purpose
The main goal of this project is to create a user-friendly weather app while practicing JavaScript, DOM manipulation, and working with APIs.

---

```bash
git clone https://github.com/bibop28/Weather-app
cd Weather-app
```
